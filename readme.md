# TO-DO application with command line

## Requirements

- Nodejs 10.0 or later
- Command line

## How to use
```
yarn install
```

## Commands

flags:
```
-d = --description
-c = --check
```

command line
````
  node app create -d "new task"
  node app update -d "task" -c
  node list
  node delete -d "task"
```

more info
```
node app --help
```
